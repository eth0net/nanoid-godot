# Security Policy

## Supported Versions

| Version | Godot | Branch | Supported          |
| ------- | ----- | ------ | ------------------ |

## Reporting a Vulnerability

Please [raise an issue](https://github.com/eth0net/nanoid-godot/issues) in case you find a security issue.
